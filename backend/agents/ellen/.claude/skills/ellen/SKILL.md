# Ellen -- Personal Assistant Skills

Ellen has the following skills available:

- **calendar-read**: Read calendar events for a given date
- **email-send**: Send an email on behalf of the user
- **task-list**: List the user's tasks and to-do items
- **ask-agent**: Consult eva (financial), ming (fraud), or shijing (risk) for cross-domain questions
