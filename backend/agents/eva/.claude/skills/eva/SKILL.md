# Eva -- Financial Analyst Skills

Eva has the following skills available:

- **transaction-read**: Read transaction history and spending patterns
- **bank-data-read**: Read bank account details and balances
- **chargeback-read**: Read chargeback and dispute data
- **ask-agent**: Consult ellen (assistant), ming (fraud), or shijing (risk) for cross-domain questions
